# 实训报告智能批改系统

基于Trae的实训报告智能批改系统，支持读取Word和PDF格式的实训报告，与参考答案进行比对分析，自动评分并生成详细的批改报告。

## 功能特点

- **多格式支持**：支持 .docx 和 .pdf 格式的文档读取
- **图片分析**：自动统计图片数量、评估图片质量
- **章节识别**：智能识别实训报告的常见结构（实训目的、实训步骤、实验结果等）
- **内容比对**：基于TF-IDF和余弦相似度的内容比对
- **自动评分**：按章节图片数量进行评分（五张图片对应五个章节，每章20分）
- **评语生成**：自动生成个性化评语和改进建议
- **批量处理**：支持批量批改整个文件夹的学生报告
- **多格式输出**：支持JSON和HTML格式的批改报告输出
- **Web界面**：基于Streamlit的交互式Web应用

## 安装说明

### 环境要求
- Python 3.8+
- 依赖库见 requirements.txt

### 安装步骤

1. 克隆或下载项目代码
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用说明

### 命令行方式

```bash
python main.py --reference <参考答案文件> [选项]
```

| 参数 | 缩写 | 说明 | 必填 |
|------|------|------|------|
| --reference | -r | 参考答案文件路径 | 是 |
| --student | -s | 学生报告文件路径（单文件模式） | 否 |
| --input-dir | -i | 学生报告文件夹路径（批量模式） | 否 |
| --output | -o | 输出文件路径（单文件模式） | 否 |
| --output-dir | -d | 输出文件夹路径（批量模式） | 否 |
| --format | -f | 输出格式：json/html/both（默认html） | 否 |

### 单文件批改

```bash
python main.py -r reference.docx -s student_report.docx -o result -f html
```

### 批量批改

```bash
python main.py -r reference.docx -i ./reports/ -d ./results/ -f html
```

### Web界面方式

```bash
streamlit run app.py
```

然后在浏览器中访问 http://localhost:8501

## 评分标准

系统按参考答案中的五张图片对应五个章节进行评分（百分制）：

| 章节 | 分值 | 说明 |
|------|------|------|
| 实训目的 | 20分 | 包含至少1张图片 |
| 实训步骤 | 20分 | 包含至少1张图片 |
| 实验结果 | 20分 | 包含至少1张图片 |
| 问题反思 | 20分 | 包含至少1张图片 |
| 心得体会 | 20分 | 包含至少1张图片 |

## 配置说明

配置文件 `配置文件.py` 包含以下可配置项：

- **IMAGES_PER_SECTION**：各章节期望图片数量配置
- **IMAGE_SCORE_PER_SECTION**：每章图片得分
- **IMAGE_THRESHOLD**：图片数量匹配阈值
- **SIMILARITY_THRESHOLD**：内容相似度阈值
- **SECTION_KEYWORDS**：章节关键词配置

## 输出格式

### HTML报告
生成美观的HTML格式批改报告，包含：
- 总分展示
- 各项得分详情
- 详细评语
- 章节比对结果
- 图片分析
- 关键词匹配

### JSON报告
生成结构化的JSON数据，便于程序处理和集成。

## 注意事项

1. 系统支持离线运行，无需网络连接
2. 建议使用docx格式以获得最佳的图片提取效果
3. PDF文件的文本提取质量取决于原始文档的扫描质量
